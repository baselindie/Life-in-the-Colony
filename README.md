# Life-in-the-Colony
Bueno este juego se trata de hormigas como su nombre mismo lo dice, donde abarcaremos varios géneros de estas mismas. Como el principal y el que estoy trabajando ahora es del género Acromyrmex, depués iré incluyendo otros tipos de género(como Odontomachus, Camponotus, Carebara Diversa, etc.).
